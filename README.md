# 🎨 Color Picker

A simple Color Picker web application built using HTML, CSS, and JavaScript. Users can change the background color by clicking on different color buttons.

## 🚀 Features

- Change the background color with a single click.
- Interactive and responsive UI.
- Built using vanilla JavaScript.
- Beginner-friendly project demonstrating DOM manipulation.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
color-picker/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📖 What I Learned

- Selecting DOM elements
- Handling click events
- Updating element styles dynamically
- Organizing a simple frontend project

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
